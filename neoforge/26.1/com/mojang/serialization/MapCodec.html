    • assumeMapUnsafe

      public static <A> MapCodec<A> assumeMapUnsafe(Codec<A> codec)
      Transforms the given Codec into a MapCodec by assuming that the result of all elements is a map.

      This MapCodec will fail to encode or decode as long as the given Codec does not return or receive a map.
