    • getDependencies

      public List<ModInfo> getDependencies(IModInfo mod)
      Returns all direct loading dependencies of the given mod.

      This means: all the mods that are directly specified to be loaded before the given mod, either because the given mod has an IModInfo.Ordering.AFTER constraint on the dependency, or because the dependency has a IModInfo.Ordering.BEFORE constraint on the given mod.
