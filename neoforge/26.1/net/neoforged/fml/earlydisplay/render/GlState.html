Class GlState

java.lang.Object
net.neoforged.fml.earlydisplay.render.GlState
public final class GlState extends Object
A static state manager for a subset of OpenGL states to minimize redundant state changes.

This class tracks the current state of various OpenGL state elements and only applies changes when necessary, reducing overhead from redundant state changes.
