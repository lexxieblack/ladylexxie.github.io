    • appendHoverText

      default void appendHoverText(FluidStack fluidStack, Item.TooltipContext context, TooltipDisplay display, Consumer<Component> builder, TooltipFlag tooltipFlag)
      Adds additional tooltip components for the given FluidStack.

      This method is invoked when building the fluid's tooltip and allows fluids to contribute custom informational lines (for example temperature, effects, or mod-specific data).

      Implementations should emit tooltip components via the provided builder consumer.

      Parameters:
      fluidStack - the fluid stack being rendered
      context - the tooltip context
      display - controls which tooltip elements should be displayed
      builder - consumer used to append tooltip components
      tooltipFlag - controls tooltip verbosity and advanced information
