  • Field Details

    • SYNCED_ATTACHMENT_TYPES

      public static final Registry<AttachmentType<?>> SYNCED_ATTACHMENT_TYPES
      Contains all entries added to NeoForgeRegistries.ATTACHMENT_TYPES with a sync handler. This ensures that non-synced attachments can be used freely on either side, but synced attachments must match across client and server. This also ensures that we can use the raw ids for network syncing.

      Should never be registered against directly. Entries are automatically added with ATTACHMENT_TYPE_ADD_CALLBACK.
