Class TagsUpdatedEvent

java.lang.Object
net.neoforged.bus.api.Event
net.neoforged.neoforge.event.TagsUpdatedEvent
Direct Known Subclasses:
TagsUpdatedEvent.ClientPacketReceived, TagsUpdatedEvent.ServerDataLoad
public sealed class TagsUpdatedEvent extends net.neoforged.bus.api.Event permits TagsUpdatedEvent.ServerDataLoad, TagsUpdatedEvent.ClientPacketReceived

Fired when tags are updated on either server or client. This event can be used to refresh data that depends on tags.

Listeners intending to use this event to update static data guarded by shouldUpdateStaticData() should listen to the root event. Listeners interested in the side-specific notification or the side-specific context should listen to the TagsUpdatedEvent.ServerDataLoad and TagsUpdatedEvent.ClientPacketReceived subclasses instead.
