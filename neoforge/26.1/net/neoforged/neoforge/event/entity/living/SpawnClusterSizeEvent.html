public class SpawnClusterSizeEvent extends LivingEvent
This event is fired from NaturalSpawner.spawnCategoryForPosition(MobCategory, ServerLevel, BlockPos) when the spawning system determines the maximum amount of the selected entity that can spawn at the same time.
