public class ItemAccessFluidHandler extends ItemAccessResourceHandler<FluidResource>
Base implementation of a simple fluid ResourceHandler backed by an ItemAccess. Data is stored in a SimpleFluidContent component.

This class allows the backing items to contain any partial level of fluid up to its capacity.
