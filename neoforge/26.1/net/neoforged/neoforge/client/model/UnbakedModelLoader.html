public interface UnbakedModelLoader<T extends UnbakedModel>
A loader for custom unbaked models.

If you do any caching, you should implement ResourceManagerReloadListener and register it with AddClientReloadListenersEvent.

      The JsonDeserializationContext argument can be used to deserialize types that the system already understands.
