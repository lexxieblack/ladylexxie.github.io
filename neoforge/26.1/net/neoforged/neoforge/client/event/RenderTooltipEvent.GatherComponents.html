Class RenderTooltipEvent.GatherComponents

public static class RenderTooltipEvent.GatherComponents extends Event implements ICancellableEvent
Fired when a tooltip gathers the TooltipComponents to be rendered, before any text wrapping or processing. The list of components and the maximum width of the tooltip can be modified through this event.

This event is cancellable, and does not

invalid reference 
have a result
. If this event is cancelled, then the list of components will be empty, causing the tooltip to not be rendered and the corresponding RenderTooltipEvent.Pre and RenderTooltipEvent.Texture to not be fired.

This event is fired on the main Forge event bus, only on the logical client.
