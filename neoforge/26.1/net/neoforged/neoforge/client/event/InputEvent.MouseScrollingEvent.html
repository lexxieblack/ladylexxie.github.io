Class InputEvent.MouseScrollingEvent

All Implemented Interfaces:
ICancellableEvent
Enclosing class:
InputEvent
public static class InputEvent.MouseScrollingEvent extends InputEvent implements ICancellableEvent
Fired when a mouse scroll wheel is used outside of a screen and a player is loaded, before being processed by vanilla.

This event is cancellable. If the event is cancelled, then the mouse scroll event will not be processed further.

This event is fired on the main Forge event bus, only on the logical client.
