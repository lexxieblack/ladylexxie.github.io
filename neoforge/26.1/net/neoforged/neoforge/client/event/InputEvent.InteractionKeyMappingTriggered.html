public static class InputEvent.InteractionKeyMappingTriggered extends InputEvent implements ICancellableEvent
Fired when a keymapping that by default involves clicking the mouse buttons is triggered.

The key bindings that trigger this event are:

  • Use Item - defaults to left mouse click
  • Pick Block - defaults to middle mouse click
  • Attack - defaults to right mouse click

This event is cancellable. If this event is cancelled, then the keymapping's action is not processed further, and the hand will be swung according to shouldSwingHand().

This event is fired on the main Forge event bus, only on the logical client.
