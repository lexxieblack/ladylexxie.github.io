Class AddDebugSubscriptionFlagsEvent

java.lang.Object
net.neoforged.bus.api.Event
net.neoforged.neoforge.client.event.AddDebugSubscriptionFlagsEvent
public final class AddDebugSubscriptionFlagsEvent extends Event
This event allows mods to register client-side debug subscription flags.

These flags are used to determine when subscriptions are allowed to register and store debug renderer values. If no flag is registered for a given debug subscription then it will be treated as a always-disabled flag and no debug values will be stored for it.

This event is fired once per tick during ClientDebugSubscriber.requestedSubscriptions().

This event is fired on the logical client.

API Note:
Each debug subscription should only be registered once.
