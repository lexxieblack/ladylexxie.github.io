public class ExtendedSlider extends AbstractSliderButton
Slider widget implementation which allows inputting values in a certain range with optional step size.
Implementation Note:
Note that AbstractSliderButton.value is the fractional progress of the slider from 0 to 1, whereas getValue() is the actual value from minValue to maxValue.
