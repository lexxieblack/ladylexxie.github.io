net.minecraft.world.item.equipment.trim.MaterialAssetGroup
public record MaterialAssetGroup(MaterialAssetGroup.AssetInfo base, Map<ResourceKey<EquipmentAsset>, MaterialAssetGroup.AssetInfo> overrides) extends Record
API Note:

It is highly recommended that the suffix parameter passed to modded MaterialAssetGroup.AssetInfos is prefixed by the mod id to avoid conflicts.

Do note that the suffix is run through Identifier.isValidPath(String), so things like colons cannot be used. You may want to separate the mod id and name using an underscore or dash.
