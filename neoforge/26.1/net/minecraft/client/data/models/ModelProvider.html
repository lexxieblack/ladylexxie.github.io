    • getKnownBlocks

      protected Stream<? extends Holder<Block>> getKnownBlocks()
      Returns a stream containing all blocks which must have their models/block states generated or empty if none are desired.

      When using providers for specific block usages, it is best to override this method returning the exact blocks which must be generated, or empty if generating only item models.

      Default implementation generates models for blocks matching the given modId.

      See Also:
      • getKnownItems()

    • getKnownItems

      protected Stream<? extends Holder<Item>> getKnownItems()
      Returns a stream containing all items which must have their models/client items generated or empty if none are desired.

      When using providers for specific item usages, it is best to override this method returning the exact items which must be generated, or empty if generating only block models (which have no respective item).

      Default implementation generates models for items matching the given modId.
